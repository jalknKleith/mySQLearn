# SQL Scripts

This repository contains a collection of SQL scripts covering various SQL concepts and operations. Each script is designed to demonstrate a specific topic and provide examples for understanding and implementing SQL queries.

**Scripts:**

- **ALTER.sql**: Illustrates the ALTER command for modifying table structures.
- **AND_OR_NOT.sql**: Demonstrates the use of AND, OR, and NOT operators in SQL queries.
- **BETWEEN.sql**: Explains the BETWEEN operator for querying data within a specified range.
- **COUNT()AVG()SUM().sql**: Covers the COUNT(), AVG(), and SUM() aggregate functions for statistical calculations.
- **CREATE.sql**: Shows the syntax and examples for creating tables in SQL.
- **DISTINCT.sql**: Explores the DISTINCT keyword for removing duplicate records from query results.
- **INSERT_INTO.sql**: Provides examples for inserting new data into existing tables.
- **LIKE.sql**: Demonstrates the LIKE operator for pattern matching and partial text searches.
- **MIN()MAX().sql**: Introduces the MIN() and MAX() aggregate functions for finding minimum and maximum values.
- **NULL.sql**: Explains the concept of NULL values and how to handle them in SQL queries.
- **ORDERBY**: Demonstrates the ORDER BY clause for sorting query results.
- **SELECT**: Provides examples for performing basic SELECT queries and specifying columns to retrieve.
- **UPDATE**: Shows how to update existing data in tables using the UPDATE command.
- **columns**: Covers the concept of creating and altering table columns.
- **constraints**: Explains the different types of constraints used in SQL to enforce data integrity.
- **dateDataType**: Demonstrates the use of date and time data types in SQL queries.
- **exIMPort**: Provides examples for exporting and importing data from and to CSV files.
- **generalSyntax**: Summarizes the general syntax and structure of SQL queries.
- **insertDataValues**: Shows examples for inserting multiple rows of data into tables.
- **keys**: Explains the concept of primary and foreign keys for establishing relationships between tables.
- **stringDataType.txt**: Demonstrates the use of string data types and string manipulation functions in SQL.
